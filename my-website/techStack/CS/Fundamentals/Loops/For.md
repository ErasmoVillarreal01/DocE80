A for loop iterates for a certain defined range.
```CS
for (initialization; Boolean; iterator) {
  //Statements
}
```

Example:
```CS
int sum = 0;
for (int i = 0; i < 10; i++){
  sum = sum + i;
}

Console.WriteLine(sum);
```

- `break` : Exits the loop
- `continue`: Continues on to the next iteration
