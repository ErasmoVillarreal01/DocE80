# Schema Compare

## What is a Schema Compare
Schema Compare in Visual Studio is a tool that lets you compare and synchronize the structure (schema) of two SQL Server databases, database projects, or DACPAC files. It highlights differences and generates scripts to make the target schema match the source
    
    1. New project
    2. SQL Server DB Project
    3. In solution. Right Click in the name (PEPSI_...) and Schema Compare
    4. From DB (PEPSI SDM) to Solution  -> Compare  -> Update
    5. In References, Add Database Reference and add master and msdb
    6. From Solution to DB -> Compare -> Update
