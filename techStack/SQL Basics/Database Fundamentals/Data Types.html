Data Types

NULL Type​

  • NULL: Absence of a value.

Numeric Data Types​

  • INT: Standard integer, typically 4 bytes.
  • SMALLINT: Smaller integer, typically 2 bytes.
  • BIGINT: Larger integer, typically 8 bytes.
  • TINYINT: Very small integer, typically 1 byte.
  • DECIMAL(p,s): Fixed-point number with p total digits and s digits after the decimal point.
  • NUMERIC(p,s): Similar to DECIMAL.
  • FLOAT: Floating-point number with single precision.
  • REAL: Floating-point number with single precision (often synonymous with FLOAT).
  • DOUBLE PRECISION: Floating-point number with double precision.

Date and Time Data Types​

  • DATE: Stores a date (year, month, day).
  • TIME: Stores a time of day (hour, minute, second).
  • DATETIME: Stores both date and time.
  • TIMESTAMP: Stores both date and time, often with timezone information and automatic updates.
  • YEAR: Stores a year value.

String (Character) Data Types​

  • CHAR(n): Fixed-length string of n characters.
  • VARCHAR(n): Variable-length string with a maximum length of n characters.
  • TEXT: Variable-length string, can store very long text.
  • NCHAR(n): Fixed-length Unicode string.
  • NVARCHAR(n): Variable-length Unicode string.
  • NTEXT: Variable-length Unicode string, can store very long text.

Binary Data Types​

  • BINARY(n): Fixed-length binary data.
  • VARBINARY(n): Variable-length binary data.
  • BLOB: Binary Large Object, for storing large amounts of binary data (e.g., images, files).

Boolean Data Types​

  • BOOLEAN: Stores true or false values (often represented as 1 or 0 in some systems).
  • BIT: Stores a single bit value (0 or 1).

Other Data Types​

  • UUID / UNIQUEIDENTIFIER: Stores a Universally Unique Identifier.
  • XML: Stores XML data.
  • JSON: Stores JSON data.
